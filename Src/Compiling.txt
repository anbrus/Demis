При компиляции программы использовались следующие программы:
1. Microsoft Visual C++ v6.0
2. Microsoft Assembler MASM v6.11
3. Microsoft HTML Help Workshop v4.74
4. Borland C++ v3.1
5. InstallShield Express v2.12

Порядок компиляции:
1. Устанавливаются все вышеперечисленные программы.
2. В среде Visual C++ открыватся проект Demis2000.dsw, устанавливается
   соответствующая конфигурация (Debug или Release).
3. Прописываются пути к ассемблеру MASM в параметрах проекта Custom Build
   для файла asmfunc.asm.
4. Редактируется файл makehelp.bat: изменяется путь к компилятору HTML Help (hhc).
5. Копируются все файлы из каталога Debug в каталог Release. Это ассемблер,
   компоновщик и всё что для них нужно.
6. Выполняется сборка (Build) проекта. Не должно быть ни ошибок ни предупреждений.
7. Выполняется саморегистрация библиотеки StdElem.dll, находящейся в каталоге
   соответствующей конфигурации (Debug или Release) командой
   regsvr32.exe StdElem.dll
8. Создаётся раздел реестра (и все родительские для него)
   HKEY_CURRENT_USER\Software\RGATA\Design Microsystems\Settings\ElementsLibraries
   В этом разделе создаётся строковый параметр с именем
   "StdElem.ElemLib" - это имя OLE-класса библиотеки,значение параметра:
   "SteElem.Element" - это имя OLE-класса элемента.
9. Можете запускать программу.
10.Для изготовления дистрибутива, с помощью программы установки откройте файл
   Demis.iwz и запустите сборку (Build).

Замечания:
1. При смене активной конфигурации, не забывайте регистрировать StdElem.dll.
2. Файл справки создаётся в каталоге конфигурации Debug, поэтому при запуске
   Release-а справка не работает. Можете просто скопировать файл справки в
   каталог Release.
3. Для сборки RunTasm и RunTlink используется Borland C++, так как Visual C++
   6-й версии уже не поддерживает DOS.
4. На использовании именно указанных версий программ не настаиваю, но с другими
   версиями, особенно более ранними, естественно, могут быть проблемы.
5. Если не можете понять как программа устроена, звоните (0855)21-33-74 (дом.),
   если всё понятно, то тем более звоните.
