Текстовый редактор

    Текстовый редактор предназначен для редактированию любых текстовых файлов размером не более 64 Кб и оптимизирован для редактирования текста программы на языке ассемблера. Для облегчения форматирования текста программы можно использовать табуляцию. Редактор оптимизирован для следующего форматирования текста: крайнее левое поле шириной 12 символов предназначено для ввода меток, имён подпрограмм, переменных и т.д. Первое в строке нажатие клавиши "Tab" перемещает курсор в поле мнемонического обозначения инструкции. Поле имеет ширину 6 символов. Следующее нажатие "Tab" переводит курсор в поле аргументов инструкции шириной 13 символов. Вслед за этим полем идёт поле комментария. Дальнейшие нажатия "Tab" перемещают курсор вправо на 4 символа.

         Для работы с текстом используется меню "Правка". С его помощью можно выполнять следующие операции:
1.     "Отмена" - отмена одного последнего действия;
2.     "Вырезать", "Копировать" и "Вставить" - соответствующие операции с буфером обмена.
3.     "Найти", "Найти далее" и "Заменить" - поиск и замена фрагментов текста.

         Выделить текст можно с помощью мыши, установив курсор мыши в начало нужного фрагмента и, нажав и не отпуская левую кнопку мыши, переместить курсор в конец фрагмента и отпустив кнопку мыши. Другой способ – с помощью клавиши "Shift". Для этого установите текстовый курсор в начало фрагмента, нажмите "Shift", и не отпуская его, переместите текстовый курсор клавишами курсора в конец фрагмента и отпустите "Shift".
