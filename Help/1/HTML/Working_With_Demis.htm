Порядок работы

    Для проектирования новой МПС, необходимо создать новый проект, ввести исходный текст программы, собрать архитектуру и, при необходимости, лицевую панель устройства, выполнить сборку программы, и отладить её с помощью отладчика.

    Для создания нового проекта, выберите пункт меню "Файл/Новый" и в появившемся списке документов выберите "Проект". Сразу после этого появится стандартный диалог сохранения файла, где необходимо выбрать папку, в которой будут расположены файлы, принадлежащие проекту, и указать имя проекта. Если всё сделано правильно, в главном окне среды появится окно нового проекта, смотри "Проект".

    Изначально проект содержит один файл текста программы с заготовкой программы и файл пустой архитектуры. Если необходимо, можно добавить в проект нужные файла с текстами программ. Файлы, которые будут участвовать в ассемблировании и компоновке программы по отдельности, размещаются в папке проекта "Ассемблируемые", включаемые файлы – в папке "Включаемые". Для добавления и удаления файлов проекта, щёлкните правой кнопкой мыши на папке или на имени файла и выберите нужное действие в появившемся меню. Меню папок позволяет добавлять файлы в проект, меню имён файлов – открывать и удалять файлы. Чтобы открыть файл, включённый в проект, достаточно сделать двойной щелчок мышью на его имени.

    Теперь необходимо настроить параметры проекта. Для этого выберите пункт меню "Проект/Параметры". В появившемся диалоге укажите объём ПЗУ новой системы и тактовую частоту процессора, смотри "Параметры проекта". Выбор тактовой частоты никак не влияет на скорость прогона программы, а используется только для расчёта временных задержек элементами архитектуры (например, время дребезга кнопки). После ввода параметров нажмите кнопку "ОК" и сохраните проект (меню "Файл/Сохранить"). Параметры проекта впоследствии можно изменить. Теперь можно приступать к вводу текста программы или сборке архитектуры.

    Изначально, проект содержит один файл текста программы. Файл содержит стандартную заготовку, которая содержит описание сегментов векторов обработчиков прерываний, данных, констант, стека и кода. Значения, которые необходимо задать в заготовке, обозначены последовательностью символов "XXXX". Эти последовательности необходимо заменить необходимыми Вам числами. Внимательно прочитайте раздел "Подготовка текстов программы.
