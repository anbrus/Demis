Параметры проекта

    Этот диалог можно вызвать через меню "Проект/Параметры", когда открыт проект. В диалоге можно задать тактовую частоту микропроцессора в герцах, объём ПЗУ МПС в килобайтах и уровень свободного входа. Выбор тактовой частоты никак не влияет на скорость прогона программы, а используется только для расчёта временных задержек элементами архитектуры (например, время дребезга кнопки).
