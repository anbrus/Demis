Проект

    Окно проекта содержит дерево всех файлов, используемых в текущей разработке.

    Ветвь "Ассемблируемые" содержит файлы, которые будут по отдельности ассемблированы в объектные файлы и затем скомпонованы (смотри раздел "Подготовка текстов программы"). Ветвь "Включаемые" содержит любые текстовые файлы, никак не обрабатываемые во время сборки. Ветвь "Архитектура" содержит единственный файл - файл архитектуры.

    Для добавления файлов в ветвь, воспользуйтесь контекстным меню, вызываемом щелчком правой кнопки мыши на имени файла. Для открытия или удаления, вызовите контекстное меню нужного файла аналогичным образом. Контекстное меню ассемблируемых файлов дополнительно содержит пункт "Содержит стартовую точку". Его можно пометить галочкой просто выбрав. Если пункт помечен, то этот файл будет скомпонован последним, что обеспечит правильное расположение стартовой точки. Подробнее смотрите раздел "Подготовка текстов программы".
